These are bundled into the browser - NEVER put secrets here.

NEXT_PUBLIC_API_URL=https://api.yourdomain.com
NEXT_PUBLIC_WS_URL=wss://api.yourdomain.com
NEXT_PUBLIC_EVO_INSTANCE_NAME=mantra_instance

NEXT_PUBLIC_ENABLE_DEVTOOLS=false
NEXT_PUBLIC_ENABLE_MOCK_DATA=false

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Failed to fetch" errors
- Check NEXT_PUBLIC_API_URL matches your actual API URL
- Verify backend is running: docker compose ps
- Check CORS configuration: FRONTEND_URL must match

Issue: WebSocket connection fails
- Ensure NEXT_PUBLIC_WS_URL uses correct protocol (ws:// or wss://)
- Check no firewall blocking port 3001
- Verify backend WebSocket endpoint is running

Issue: 401 Unauthorized errors
- Check JWT_SECRET is set and consistent
- Verify you're logged in (check browser cookies)
- Ensure JWT token hasn't expired

Issue: Database connection failed
- Check DATABASE_URL format
- Verify PostgreSQL is running: docker compose logs postgres
- Check credentials match POSTGRES_USER and POSTGRES_PASSWORD
